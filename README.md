# README

# Dependency

* Ruby version: 3.3.0

## Configuration

```bash
rails db:migrate
rails tailwindcss:install
rails db:seed
```

## Start Server

- visit: `http://127.0.0.1:3000/` to get the home page.
- visit: `http://127.0.0.1:3000/admin` with `admin@example.com/password` to login the admin page.
